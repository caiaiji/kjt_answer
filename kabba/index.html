2如何搜索题目？

  • 在顶部搜索框输入 题号（如 1001）、题目名称（如 排序）或 描述关键词
  • 支持按 难度 筛选：入门 / 普及- / 普及/提高- / 普及+/提高 / 提高+/省选- / 省选/NOI-
  • 支持按 标签 筛选：点击标签栏的标签可多选
  • 支持按 通过率 / 提交数 排序
  • 用 代码筛选 只看有参考代码的题目

3如何查看和复制代码？

  • 点击任意题目打开详情弹窗
  • 弹窗底部显示 参考代码（暗色代码框）
  • 点击右上角 「复制代码」 按钮一键复制
  • 弹窗打开后可用 ← → 键切换上/下一题
  • 按 Esc 关闭弹窗

小提示：按 / 键可快速聚焦到搜索框，无需鼠标点击。列表中有高亮齿轮图标（⚙）的题目表示已有参考代码。
